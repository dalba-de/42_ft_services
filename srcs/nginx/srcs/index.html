Welcome to Fantabulous World of Kubernetes

Kubernetes

This is a System Administration and Networking project.

Ft_services will introduce you to kubernetes. You will discover cluster management and deployment with Kubernetes. You will virtualize a network and do "clustering".

Mandatory

The project consists of setting up an infrastructure of different services. To do this, you must use Kubernetes. You will need to set up a multi-service cluster. Each service will have to run in a dedicated container. Each container must bear the same name as the service concerned and For performance reasons, containers have to be build using Alpine Linux. Also, they will need to have a Dockerfile written by you which is called in the setup.sh. You will have to build yourself the images that you will use. You will also have to setup:
  • "The Kubernetes web dashboard."
  • "The Load Balancer which manages the external access of your services."
  • "A Nginx server listening on ports 80 and 443."
  • "A FTPS server."
  • "A WordPress website."
  • "PhpMyAdmin linked with the MySQL database."
  • "A Grafana platform, listening on port 3000, linked with an InfluxDB database."
